# Proyecto de Infraestructura en la Nube
 
MIEMBROS DEL EQUIPO
- Luis Maroto
- Vanessa Hernandez
- Karina Corrales

Antecedente
Un grupo de becarios del programa AWS Re/Start asumió el desafío de crear un proyecto final que consolidara los conocimientos adquiridos durante 5 meses de capacitación. El objetivo era no solo aplicar las habilidades técnicas aprendidas, sino también simular un entorno de trabajo que se asemejara lo más posible a una empresa de tecnología. De esta manera, se buscaba poner a prueba incluso las habilidades blandas, como la comunicación y el liderazgo.

A lo largo del proyecto, surgieron diversos desafíos, como cambios en las prioridades de los miembros que formaron parte de la primera convocatoria del proyecto, la necesidad de reequilibrar las cargasde trabajo y ajustar el alcance del proyecto. Además, se exploraron herramientas de AWS que el tiempo del programa Re/Start limitó a mencionar. Todo esto se llevó acabo para lograr un desarrollo colaborativo del proyecto de forma remota y ágil.
